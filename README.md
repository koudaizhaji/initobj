## 🥈环境要求

> - node推荐16.18.0（>14.18.0以上）
> - pnpm 为工程提供包管理服务。
> - Vue 作为组件库的框架基础。
> - Vite 作为主力构建工具。
> - TypeScript 提供类型支持。
